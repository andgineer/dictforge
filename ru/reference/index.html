    dictforge.export_stardict.StarDictExportFormat

    Bases: ExportFormat

    Export dictionary to StarDict format.

    StarDict is an open format used by various dictionary applications. It consists of three main files: - .ifo: Dictionary information/metadata - .idx: Word index with offsets and sizes - .dict/.dict.dz: Dictionary data (optionally compressed)
